ARNAV GUPTA PRESENTS AT MOBILE AND DISRUPTIVE TECHNOLOGY SUMMIT
Personalized Google Home
Coding Blocks founding member Arnav Gupta presented a talk titled Build your own Google Home using Raspberry Pi and Android Things. In his talk, Arnav used a Raspberry Pi 3 running Google Things OS, which is Google's extremely stripped-down Android version for IOT devices. The talk covered two major Android Things APIs: Bluetooth Audio API to enable Raspberry Pi to be paired with any Bluetooth speaker and speak using that; and the Android Assistant API to answer the queries which basically gives the same results that Google Home gives. Through this talk, developers learned how to make their own Google Home systems.
VIEW DECK
Push to Play: End-to-end CI/CD
Coding Blocks founding member Arnav Gupta presented a talk titled Push to Play: End-to-End CI/CD Setup using Zero Infrastructure. Using three free-to-use tools/services, namely Github, Travis-CI and Google Play Developer APIs, Arnav talked about automating the entire process. The talk dealt with Android Testing & Deployment lifecycle, including CI Setup, Unit Tests, Instrumentation Tests, UI Tests and wrapping them up in test suites and harnesses and getting coverage reports. Arnav also showed a demo of the available tools to continuously integrate tests like Travis CI, Codacy and Codecov.io or Coveralls.
