CODE-N-COUNTER 2.0
After a very successful first edition last year, Nagarro and Coding Blocks came together again in 2017 to organize Code-N-Counter 2.0, the second edition of Nagarro's annual coding hackathon. The online round was held on HackerEarth in August and saw a participation of over 4500 teams. The top 100 teams were called for the final round to be held at the Nagarro HQ in Gurugram on Sept 2 and 3. The two day finale saw teams from various corners of the country, with ingenious ideas and implementations. The event was indeed a success.
OPEN SOURCE AND GSOC MEETUP AT DWARKA
Coding Blocks conducted an exclusive meetup for Open Source and GSoC enthusiasts at our Dwarka centre on Sept 10. The event saw a variety of speakers speak on topics such as the importance of open source, various avenues of open source development, GSoC experiences and also various GSoC organizations. The highlight of the day was a live chat with Mario Behling and HP Dang from FOSSASIA. The valediction ceremony for BOSS also took place and all prize winners talked about their experiences. The event was attended by over 100 open source developers and enthusiasts, who thoroughly enjoyed the event.

WORKSHOPS AND BOOTCAMPS ORGANIZED
APP DEVELOPMENT AND DEPLOYMENT
Coding Blocks organized a workshop on App Development and Deployment using Java and Gradle at the Greater Noida centre. The workshop was conducted by GSoC mentor Anuj Garg on Sept 2. It was intended towards getting students started into application development using Java. The mentor covered topics such as dependency management, Gradle and some third-party libs to make a small http server and VLC java wrapper interface. Students also made a VLC remote for LAN.
FACEBOOK CHATBOTS USING JAVA
Coding Blocks organized a workshop on Facebook Chatbot using Java at the Greater Noida centre. The workshop was conducted by GSoC mentor Anuj Garg on Sept 3. It was intended towards introducing students to REST API and its real-life usage with Facebook chatbots. Topics such as Spring Boot, Facebook Webhook and Messenger API were discussed; students deployed their code to Heroku for launching and iterating their project; and eventually made a Joke Teller Chatbot.
ANDROID GAME DEVELOPMENT
Coding Blocks organized a three-day bootcamp on Android Game Development at the  Greater Noida centre. The bootcamp was conducted by GSoC mentor Anuj Garg from Sept 30 to Oct 2. It introduced students to topics such as popular Java game framework LibGdx, and covered Score HUD, Parallax effect, viewports, physical world simulation and Admob for monetising. Students also made a Mario like game with Jungle UI at the end.
GAME DEVELOPMENT AT JIIT NOIDA
Coding Blocks organized a two-day workshop on Game Development at Jaypee Institute of Information Technology, Sector 62, Noida. The workshop was conducted on Sept 27 and 28 by web and game development expert, Harpreet Singh. It introduced students to basic game development technologies such as HTML 5 and Javascript. The students also learnt advanced game development frameworks Phaser and Canvas.
