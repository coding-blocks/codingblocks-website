Overview



Coding Blocks in association with Nagarro is organizing this online bootcamp on Game Development. It is an opportunity for everyone who is interested in learning how to build games using HTML5, Javascript and Phaser which is a popular game development framework. In bootcamp, instructor will cover games such as Pokemon Shooter, Mario, Flappy Birds and many more.

The bootcamp will be conducted online which means you don't need to travel anywhere to attend classes. You can join sessions online using your laptop. Apart from Game Development, this bootcamp will introduce you to the world of JavaScript and HTML5 which is most essential skill in the technology world.
There will be total 5 sessions in the bootcamp. Sessions will be conducted on following dates : April 8th, 9th, 12th, 15th and 16th. Timings of the session would be from 4 pm to 7.30 pm.

HTML5 Canvas

The HTML Canvas element is used to draw graphics, on the fly, via JavaScript. The canvas element is only a container for graphics. You will learn how to play with canvas, draw graphics and render games.

JavaScript | Phaser

In the bootcamp, you will learn the basics as well as advanced topics like implementing event listeners, object oriented programming in JS. You will also learn how to use Phaser framework to build uber cool games

Multilevel Games

Not just single state games, you will also learn to create multi-level games using Phaser State Managar and switch between them.

Physics Engine

Most Arcade Games require some kind of physics to be implemented in the game. In this bootcamp we will explore the Phaser's Arcade Physics engine to add gravity, velocity and simulate collisions on our objects.
